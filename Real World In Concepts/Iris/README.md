# 🌸 Iris Classifier mit PyTorch

Dieses Projekt zeigt ein vollständiges Beispiel, wie man ein neuronales Netz mit PyTorch trainiert, evaluiert, speichert und wieder lädt – basierend auf dem bekannten Iris-Datensatz.

## 🔍 Funktionen

- Klassifiziert Iris-Blumenarten (Setosa, Versicolor, Virginica)
- Nutzt einen einfachen Feedforward-Neural-Network-Ansatz
- Enthält Training, Evaluation und Test-Aufteilung
- Speichert Modell & führt Vorhersagen durch
- Visualisiert Training Loss, Eval Loss und Test Accuracy

## 🧠 Verwendete Technologien

- Python 3.10+
- PyTorch
- scikit-learn
- matplotlib
- numpy
